# Converting-YOLOv8-to-.hef
This repository provides a guide on how to convert a YOLOv8 model to a .hef file so it can run on Hailo Hardware Accelerators


To being get your .pt YOLOv8 model.

If you are using a custom model you will also need the images you trained your model on in a folder.

To begin you will need a Linux system, Virtual Machine, or WSL with the following specifications:
-Ubuntu 20.04/22.04, 64-bit
-16+ GB RAM
-Python 3.8/3.9/3.10

For a step by step guide go to the step-by-step page.
